# 党员、干部信息智能管理 PRD

## 目标

围绕党群工作部党员、干部信息治理场景，建设具备自动核验、重复清洗、完整性检查、动态更新提醒和治理留痕能力的应用，提升人员台账维护效率并降低信息错误率。

## 核心模块

1. 工作概览：集中呈现台账总量、差异项、质量问题、提醒任务和近期审计留痕。
2. 人员台账：统一展示姓名、组织、职务、党籍状态、学历、联系方式等主数据。
3. 自动核验：对导入批次与存量数据进行字段比对，生成差异清单和核验建议。
4. 重复清洗：识别同音姓名、身份证号异常、组织名称口径不一等疑似重复问题。
5. 完整性检查：按必填字段规则输出缺失项清单、缺失分布和补全建议。
6. 动态提醒：围绕岗位变动、状态变更、材料补录和月度复核等事件触发提醒。
7. 治理底座：提供登录、用户管理、审计日志、系统设置和 AI 对话能力。

## 验收要点

- 首页可登录并进入系统。
- 至少包含 6 个业务页面和 3 个治理页面。
- Prisma 数据模型覆盖主数据、导入核验、质量问题、提醒任务和会话记录。
- 种子数据可初始化完整演示场景。
- `db:init / typecheck / lint / build / verify` 全部通过。
